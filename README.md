# 🚢 Titanic Survival Analysis

## 📌 Project Overview
This project performs an in-depth Exploratory Data Analysis (EDA) on the Titanic dataset to uncover the key factors that influenced passenger survival. Using Python, data visualization, and Power BI, the project reveals meaningful patterns behind one of history’s most famous disasters.

---

## 📊 Dataset Information
- Source: Kaggle Titanic Dataset  
- Total Passengers: 891  
- Features: 12  
- Survival Rate: 38.4%  

---

## 🎯 Objectives
- Analyze survival patterns based on gender, class, age, and family size  
- Handle missing data and perform feature engineering  
- Create meaningful visualizations for storytelling  
- Generate insights for predictive modeling  

---

## 🛠 Tools & Technologies Used
- Python (Pandas, NumPy)  
- Matplotlib & Seaborn  
- Jupyter Notebook / Google Colab  
- Power BI (Dashboard)

## 🔍 Key Insights

### 👩 Gender Impact
- Female Survival Rate: **74.2%**
- Male Survival Rate: **18.9%**
- 👉 Strongest predictor of survival

### 🎫 Passenger Class Impact
- 1st Class: **63% survival**
- 2nd Class: **47% survival**
- 3rd Class: **24% survival**
- 👉 Higher class = higher survival chances

### 🎂 Age Impact
- Children (0–12): **58% survival**
- Adults: ~**38% survival**
- Seniors (60+): **22.7% survival**
- 👉 Children were prioritized

### 👨‍👩‍👧 Family Size Effect
- Small families (2–4): Best survival (~57%)
- Solo travelers: ~30%
- Large families: Lower survival
- 👉 Balance between support & coordination

## 🧹 Data Cleaning & Feature Engineering
- Missing Age values filled using median (Pclass + Sex)  
- Missing Embarked filled with mode  
- Created new features:
  - AgeBucket  
  - FareGroup  
  - FamilySize  
  - IsAlone
